# EduAI
## Description
Edu AI is an intelligent educational platform designed to simplify and personalize the process of learning. After registering on the platform, users can create their own courses by choosing a subject or defining a learning goal. Instead of searching for scattered resources, learners receive a fully structured course generated by artificial intelligence.

The course creation process starts with an AI-generated learning plan that outlines the key topics, progression, and objectives. Once the plan is approved, Edu AI automatically generates detailed lectures for each topic, written in a clear and student-friendly format. To reinforce understanding, each lecture is accompanied by quizzes that allow users to test their knowledge and track their progress.

Edu AI aims to make education more accessible, efficient, and adaptive by providing personalized learning paths, interactive content, and instant feedback. The platform is suitable for self-learners, students, and anyone who wants to quickly gain new knowledge in a structured and practical way.


## API DOCUMENTATION
### Authentication API

Register<br><br>
POST /auth/users<br>

Login<br><br>
POST /auth/login<br>

Get all users (admin)<br><br>
GET /auth/users<br>

Get user by ID<br><br>
GET /auth/users/:id<br>

Update user<br><br>
PUT /auth/users/:id<br>

Delete user (admin)<br><br>
DELETE /auth/users/:id<br>

### Course API
Create course <br><br>
POST /courses<br>

Initialize course<br><br>
POST /courses/init<br>

Get all courses<br><br>
GET /courses<br>

Get course by ID<br><br>
GET /courses/:id<br>

Update course<br><br>
PUT /courses/:id<br>

Delete course<br><br>
DELETE /courses/:id<br>

### Chapters API
Add chapter<br><br>
POST /courses/:id/chapters<br>

Get chapter<br><br>
GET /courses/chapters/:chapterId<br>

Update chapter<br><br>
PUT /courses/:id/chapters/:chapterId<br>

Delete chapter<br><br>
DELETE /courses/:id/chapters/:chapterId<br><br>

Mark chapter complete<br><br>
POST /courses/chapters/:chapterId/complete<br>


### AI Generation API
Generate single chapter <br><br>
POST /ai/:id/chapters/ai<br>

Generate course plan<br><br>

POST /ai/:id/chapters/ai/batch<br>

Confirm plan & generate full course<br><br>
POST /ai/:id/confirm<br>
